# Known Limitations

## Audience

This document is for users and beta testers.

- The current private beta path is Windows-only.
- Distribution is still limited to private beta participants rather than a general public release.
- The optional local AI assistant is not required for the main workflow and may be unavailable on some machines.
- Large ChatGPT exports can still take significant time to process even when the workflow is healthy.
- Very large workspaces can take extra time to open secondary screens after import. `Find Imports` is usually the fastest immediate follow-up, while `Readable Archive` can take a few minutes to finish loading when the workspace contains many thousands of readable slices.
- `Datasets` should currently be treated as downstream of `Readable Archive` for very large workspaces rather than an equally fast first follow-up screen.
- Activity History is improving, but broader cross-screen action coverage is still being expanded.
- Quantum is intentionally focused on supported AI export packages rather than broad general document ingestion.
- Some advanced tools remain oriented toward maintainers or power users and are not the main beta workflow.
- `Ask Quantum` is still experimental and should not be treated as the authoritative path when it conflicts with deterministic archive, dataset, or activity-history outputs.
- `Diagnostics` artifacts may exist on disk even when the packaged app UI does not yet open or summarize them correctly.
