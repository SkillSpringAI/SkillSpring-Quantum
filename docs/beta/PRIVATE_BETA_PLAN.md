# Private Beta Plan

## Target

Begin a small private beta by **15 August 2026**.

## Beta purpose

Validate whether general users can:

1. launch Quantum
2. choose the right export
3. understand the preflight result
4. complete or safely stop an import
5. review archive output
6. review dataset output
7. search prior imported material

## Initial cohort

1. one cooperative general-use tester with an ordinary export
2. one tester with a different conversation style or vendor
3. one more formal professional-context tester after the first support issues are understood

## Pre-beta gates

- reproducible Windows packaging path
- tester guide
- known limitations note
- feedback template
- internal release rehearsal

## What counts as success

- testers can complete the main workflow without live scripting help
- failures are diagnosable from local artifacts
- the deterministic workflow remains useful even without optional local AI
