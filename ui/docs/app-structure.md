# App Structure

## Core layout

### Left sidebar
- Dashboard
- Imports
- Organized Output
- Datasets
- Tiered DB
- Review Queue
- Diagnostics
- Governance
- Settings

### Top bar
- App name
- current run status
- import button
- search
- notifications / settings

### Main pane
Primary content area for the selected section.

### Detail pane
Context and actions for the selected record or item.

## Recommended feel
- desktop utility first
- minimal clutter
- strong information hierarchy
- table + detail workflow
- log visibility where useful
- easy access to manifests and diagnostics
