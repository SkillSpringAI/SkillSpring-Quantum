# Known Limitations

## Audience

This document is for users and beta testers.

- The current private beta path is Windows-only.
- Distribution is still limited to private beta participants rather than a general public release.
- The optional local AI assistant is not required for the main workflow and may be unavailable on some machines.
- Large ChatGPT exports can still take significant time to process even when the workflow is healthy.
- Activity History is improving, but broader cross-screen action coverage is still being expanded.
- Quantum is intentionally focused on supported AI export packages rather than broad general document ingestion.
- Some advanced tools remain oriented toward maintainers or power users and are not the main beta workflow.
