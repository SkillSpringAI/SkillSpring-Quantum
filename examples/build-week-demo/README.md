# Build Week Demo Export

This folder contains a small synthetic ChatGPT-style export for evaluating SkillSpring Quantum without using personal conversation history.

## Use it

1. Open Quantum and select `Imports`.
2. Choose `chatgpt-conversations.json` from this folder.
3. Run `Export Check` and then import the same file.
4. Open `Readable Archive`, `Datasets`, or `Find Imports`.

In `Find Imports`, start with one of Quantum's suggested topics and refine from the visible imported results if needed.

## Privacy

Every conversation in this export is synthetic. It is included only to demonstrate the import, archive, dataset, and retrieval workflow.
