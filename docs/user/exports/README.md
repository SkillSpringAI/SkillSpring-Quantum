# Export Guides

Use these guides before running a full import in Quantum.

The ordinary beta workflow is:

`Imports -> Readable Archive -> Datasets -> Find Imports`

Start in `Imports`, run `Export Check`, and import from the same file or folder only after the check matches the vendor export you expected.

## Supported Guides

- [ChatGPT](CHATGPT.md)
- [Claude](CLAUDE.md)
- [Gemini](GEMINI.md)
- [Microsoft Copilot](COPILOT.md)
- [Grok](GROK.md)

## What Quantum Expects

Quantum expects the downloaded export file or the extracted export folder, depending on the vendor and export shape. If you are unsure, select the top-level downloaded `.zip`, `.json`, `.html`, or `.csv` file first and run `Export Check`.

Do not import private, employer, supplier, customer, or family data unless you are comfortable reviewing it locally and have permission to use it.
