# Governance Logs

This folder exists to track governance-layer changes over time.

Use it to record:
- rule changes
- threshold changes
- policy changes
- dataset handling changes
- diagnostics standard changes

## Intended use

- change-log.md -> running log of meaningful governance changes
- rule-change-template.md -> reusable template for documenting a single rule update

## Principle

If rule behavior changes, it should be documented here even when the code still runs.
